I recently rebuilt my portfolio using [Eleventy](https://www.11ty.dev/), a simple and flexible static site generator. Here's what I learned along the way.

## What is Eleventy?

Eleventy (or 11ty) is a static site generator that turns templates and data into HTML. It supports many template languages — I'm using Nunjucks and Markdown.

## What I liked

- **Simple config** — the `.eleventy.js` file is easy to understand.
- **Collections** — tagging content lets you group and list pages easily.
- **Data files** — JSON/JS files in `_data` are automatically available in templates.

## Challenges

- Understanding the directory data file system took a bit of reading.
- Pagination is powerful but requires some trial and error.

Overall, Eleventy is a great choice for a developer portfolio. Highly recommend it!
